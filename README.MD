## Hyperline-aio

### Install `Node.js`
```sh
sudo apt install git
sudo apt install nodejs
sudo apt install npm
node -v
npm -v
```
### Clone repo and install dependencies
```sh
git clone https://github.com/whonion/ts-hyperline-aio.git
cd ts-hyperline-aio
npm i
npm i -g typescript
```
### Run script
```sh
npm run start
```
